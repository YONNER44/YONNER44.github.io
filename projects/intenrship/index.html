INTENRSHIP

Sistema de gestión de alojamiento y pasantías empresariales

Septiembre 2025 - Actualidad

Descripción del Proyecto

INTENRSHIP es un sistema integral basado en permitir el registro de diferentes empresas que quieren brindar el servicio de alojamiento alquiler de vehículos y realizar pasantías en diferentes empresas registradas.

Asociación

Proyecto desarrollado en colaboración con Disruptive Information Technologies

Características Principales

Registro de Empresas

Sistema completo para el registro y validación de empresas prestadoras de servicios

Gestión de Alojamiento

Módulo para administrar servicios de alojamiento y alquiler de vehículos

Sistema de Pasantías

Plataforma para conectar empresas con estudiantes y gestionar pasantías

Panel Administrativo

Dashboard completo para supervisión y gestión de operaciones

Sistema de Reservas

Funcionalidad para gestionar reservas de servicios

Autenticación y Permisos

Sistema robusto de roles y permisos de usuario

Stack Tecnológico

React TypeScript Tailwind CSS Node.js Express.js Figma Jira GitHub

Logros Destacados

  • Arquitectura escalable implementada para soportar múltiples empresas
  • Interfaz intuitiva y responsiva para una experiencia de usuario óptima
  • Sistema de búsqueda y filtrado avanzado
  • Integración exitosa con APIs de terceros
  • Optimización de rendimiento para grandes volúmenes de datos
