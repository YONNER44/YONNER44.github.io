Sistema de recolección automática de precios de medicamentos vía WhatsApp Web. Los proveedores envían sus listas de precios (texto o imagen) y el sistema las extrae con IA, las organiza y las sincroniza automáticamente en Google Sheets y Excel. Arquitectura containerizada con Docker Compose, backend FastAPI, frontend React y sincronización en tiempo real.
